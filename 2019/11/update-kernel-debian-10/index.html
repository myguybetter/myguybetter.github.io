Debian 10环境下更新内核
发布于：2019-11-16 20:36:41 标签：/ linux / 访问：

1. 添加Backports源

  • 写入apt源：sh -c 'printf "deb https://deb.debian.org/debian buster-backports main" > /etc/apt/sources.list.d/buster-backports.list'
  • 更新apt：apt update

2. 下载内核文件

  • 下载：apt -t buster-backports install linux-image-amd64 linux-headers-amd64 –>其他系统自行更改系统代号
  • 更新grub：update-grub
  • 重启系统：reboot

3. 检查新内核是否适用

  • 键入命令uname -r，若版本号为新内核版本即适用

4. 删除旧内核

  • 检查系统内核：dpkg -l | grep linux
  • 删除旧内核（headers和image）：apt purge ...

删除完毕后再次测试是否正常启动系统。

启用bbr

  • 键入命令：echo "net.core.default_qdisc=fq" >> /etc/sysctl.conf和echo "net.ipv4.tcp_congestion_control=bbr" >> /etc/sysctl.conf
  • 启用：sysctl -p
  • 检查：lsmod | grep bbr，有bbr模块即启用成功。
